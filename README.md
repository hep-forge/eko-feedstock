Feedstock for [eko](https://eko.readthedocs.io/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Evolution Kernel Operators

## Install

```bash
conda install -c hep-forge -c conda-forge eko
```
